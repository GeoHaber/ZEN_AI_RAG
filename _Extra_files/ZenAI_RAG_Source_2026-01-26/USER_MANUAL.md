# ZenAI User Manual

Welcome to **ZenAI**! This manual will help you navigate the application, manage your local AI models, and utilize advanced features like generic RAG (Retrieval Augmented Generation) and Swarm Intelligence.

---

## 🚀 Getting Started

### Launching the App
1.  Run the application using the launcher script or `python zena.py`.
2.  The interface will open in your default web browser at `http://localhost:8080`.

### The Interface at a Glance
-   **Left Sidebar (Drawer)**: Main navigation and controls.
-   **Center Screen**: The chat interface where you interact with the AI.
-   **Bottom Bar**: Input area for text, voice, and file attachments.
-   **Top Right**: Connection status and Settings.

---

## 📂 Main Menu (Left Sidebar)

The sidebar is your command center. You can open/close it using the menu icon `☰` in the top left.

### 1. 💬 New Chat
-   **Action**: Clears the current conversation context and starts fresh.
-   **Use Case**: Switch topics completely or reset the AI's memory.

### 2. 🤖 Model Manager
Expand this section to view and manage your local LLMs.
-   **Catalog**: Shows a list of popular models (e.g., Llama 3, Mistral, Qwen).
-   **Model Cards**: Each card displays:
    -   **Specs**: Parameter size, RAM requirement, and quantization level.
    -   **Ratings**: Speed vs. Quality trade-off.
    -   **Status**: Indicates if the model is installed or needs downloading.
-   **Custom Download**: Enter a specific Hugging Face repository and filename to download specialized models not in the default catalog.

### 3. 🧠 Neural Features
Advanced controls for how the AI processes your requests.
-   **Enable Swarm Mode (CoT)**: Activates "Chain of Thought" reasoning. The AI will "think" out loud and break down complex problems before answering. *Note: Slower but more accurate.*
-   **Smart Routing**: (If enabled) Uses a lightweight "Butler" model to handle simple queries quickly, while routing complex tasks to a larger "Master" model.

### 4. 📚 RAG / Knowledge Base
Connect ZenAI to your own data.
-   **Source Selection**: Choose where ZenAI looks for answers:
    -   **Filesystem**: Scan a local folder (PDFs, text files, code).
    -   **Website**: Crawl a specific URL.
-   **Scanning**: Click "Scan Now" to index new content. This creates a searchable database of your documents.

## 💬 Chat Interface

### Sending Messages
-   Type your message in the bottom input bar and press **Enter** or click the **Send** button.
-   **Streaming**: The AI response will appear in real-time.

### 📎 Attachments
-   Click the **Paperclip** icon to upload files.
-   **Supported Formats**: PDF, TXT, Python, Markdown, Images (OCR is experimental).
-   The content of the file is extracted and added to the context of your next message.

### 🎙️ Voice Input (TTS & STT)
-   **Speak**: Click the **Microphone** icon to record your voice. ZenAI will transcribe it to text.
-   **Listen**: Click the **Speaker** icon next to an AI response (or the "Speak" button in the menu) to hear the answer read aloud.

### ✨ Quick Actions
-   **Guided Chips**: Helpful suggestion chips (e.g., "Summarize this", "Explain code") appear above the chat bar for quick interactions.

## 🔍 Understanding RAG (Transparency Law)

ZenAI follows a "Transparency Law" to let you know exactly where information comes from.

-   **Blue Bubbles**: Standard AI responses based on its internal training.
-   **Citations [1]**: When RAG is active, the AI will cite sources using bracketed numbers.
-   **Sources Footer**: At the bottom of a RAG response, expand the **Sources** section to see:
    -   Source Title & URL/Path.
    -   Relevance Score (e.g., "95% match").
    -   A snippet of the exact text used.

## ⚙️ Settings

Click the **Gear Icon** ⚙️ in the top right of the sidebar.
-   **Dark Mode**: Toggle between Light (Day) and Dark (Night) themes.
-   **Language**: Change the interface language.
-   **Reset Config**: Restore default settings if things look weird.

## ❓ Troubleshooting

| Issue | Solution |
|-------|----------|
| **"Model failed to load"** | Check if you have enough RAM. Try a smaller model (e.g., Phi-3 or Llama-3-3B). |
| **"No audio device"** | Ensure your microphone is connected and permissions are granted to the browser/terminal. |
| **"RAG found no results"** | Try re-scanning your directory or adjusting the search terms. Verify files are text-readable (not scanned images). |
| **Slow Performance** | Disable "Swarm Mode" for faster, simpler answers. |
