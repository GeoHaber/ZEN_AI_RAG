# Emotional Audio Lab
Prototype environment for integrating "Emotional" TTS using Qwen2-Audio or EmotiVoice.

## Goal
To generate speech with specific emotional inflections (e.g., [Excited], [Sad], [Whisper]) dynamically from text.

## Models Under Consideration
1. **Qwen2-Audio**: Multimodal model capable of audio understanding and generation.
2. **EmotiVoice**: Lightweight TTS engine with emotion support.
3. **F5-TTS**: Fast, high-quality TTS.

## Structure
- `voice_test.py`: Main prototype script.
- `generated/`: Output folder for .wav files.
